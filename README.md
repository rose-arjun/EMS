# Employee Management System

## 📌 Description
This is a Java-based Employee Management System developed using Spring Boot. It allows users to manage employee details such as adding, updating, viewing, and deleting employee records.

## 🚀 Technologies Used
- Java
- Spring Boot
- MySQL
- Maven

## ✨ Features
- Add new employee
- Update employee details
- Delete employee
- View employee list

## 📂 Project Structure
- controller
- service
- dao
- entity

## ▶️ How to Run
1. Clone the repository
2. Open in IDE (Eclipse / IntelliJ)
3. Configure database (MySQL)
4. Run the application
